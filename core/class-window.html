Module libnpw.core
Package com.nullpointerworks.core
Class Window
A container for a javax.swing.JFrame which serves as the primary window handler. The Window does not extend any other subclass. To gain access to the underlying frame use; window.getFrame();

void swap(int[] pixels)
Swap the integer content of the given array to the display buffer. It's up to the user to make sure that the length of the array matches the area of the display.Each integer is assumed to follow ARGB order at a bit depth of 8 bits.
Parameters:
pixels - the array of integers that represent colors

void addWindowListener(WindowListener windowlistener)
Add a java.awt.event.WindowListener to the internal frame. When creating a new window, all window listeners will be added to the new window. This method is intended to be used by the libnpw.game library which has a LoopAdapter that track the state of the application and inform the window.
