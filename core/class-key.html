Package com.nullpointerworks.core.input
Class Key
Contains static some integer ASCII members which are of common usage in games. This is not a comprehensive collection of all ASCII/UTF character codes, just the ones that are common such as letters (A-Z) and numbers (0-9), some control keys (CTRL, ALT, ESC, etc.), arrow keys and F1-10 keys.

static boolean isLetter(int code)
Returns true if the given key code is a letter in the alphabet. The ASCII codes for letters is from 65 until and including 90 for upper case letters, and from 97 until and including 122 for lower case letters.
